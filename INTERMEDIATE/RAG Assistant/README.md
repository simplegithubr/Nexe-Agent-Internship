# RAG Assistant - Document Q&A System

A powerful Retrieval-Augmented Generation (RAG) system that allows you to upload documents and ask questions to get contextual answers powered by AI.

## Features

- 📤 Upload PDF, DOCX, and TXT documents
- 🔍 Semantic search with ChromaDB vector store
- 🤖 AI-powered answers using OpenRouter's free LLaMA 3.1 model
- 💬 Interactive chat interface
- 📊 Document management and tracking

## Usage

1. Upload your documents (PDF, DOCX, or TXT)
2. Ask questions about the content
3. Get AI-generated contextual answers

## Tech Stack

- Backend: Flask, ChromaDB, Sentence Transformers
- Frontend: Vanilla JavaScript with modern CSS
- AI Model: LLaMA 3.1 8B via OpenRouter (free tier)

Check out the configuration reference at https://huggingface.co/docs/hub/spaces-config-reference
