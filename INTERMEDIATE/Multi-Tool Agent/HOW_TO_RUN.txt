2. Replace "your_openrouter_api_key_here" with your actual API key
3. Get free API key from: https://openrouter.ai/keys

Example:

    TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Port 5000 already in use
Solution: Edit backend\app.py, change port to 5001

Problem: Module not found
Solution: cd backend && pip install -r requirements.txt

Problem: OpenRouter not working
Solution: Check API key in backend\.env file
