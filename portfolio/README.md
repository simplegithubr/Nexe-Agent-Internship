## 🔧 Local Development

1. Clone the repository:
```bash
git clone https://github.com/simplegithubr/Nexe-Agent-Internship.git
cd Nexe-Agent-Internship/portfolio
```

2. Open in browser:
```bash
# Using Python
python -m http.server 8000

# Or simply open index.html in your browser
```

3. Visit `http://localhost:8000`

## 🌐 Deployment

### Deploy to Vercel (Recommended)

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Deploy:
```bash
cd portfolio
vercel
```

3. Follow the prompts

### Deploy to Netlify

1. Drag and drop the `portfolio` folder to Netlify
2. Or connect your GitHub repository

### Deploy to GitHub Pages

1. Push to GitHub
2. Go to Settings → Pages
3. Select branch and `/portfolio` folder
4. Save

## ✏️ Customization

### Update Project Links

Edit `index.html` and replace:
- `YOUR_USERNAME` with your Hugging Face username
- `your.email@example.com` with your email

### Change Colors

Edit `style.css` and modify CSS variables:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
}
```

### Add More Projects

Add new project cards in the `projects` section of `index.html`.
